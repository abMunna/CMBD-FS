STRING METHODS :

.length -> counts the number of character in a String .concat() -> adds two String .substring(a,b) -> takes a portion of a string, positive and negative index both work .indexOf(word) -> shows the index no of a word in a String .charAt(a) -> shows the character of a specific index value where the index is a .includes(word) -> checks the string to see if it includes a word .toLowerCase() .toUpperCase()
