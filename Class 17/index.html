NUMBER METHODS :

tofixed(x) -> shows a fixed number of decimal place. typeof -> shows the type of value (string, int, float etc) Math.round() -> rounds a number to a whole number Math.ceil() -> rounds a number to the next whole number, 24.1 -> 25 / 24.5 -> 25 Math.floor() -> rounds a number to the previous whole number, 24.1 -> 24 / 24 -> 25 Math.abs() -> changes values to positive parseInt() -> coverts a float to int parseFloat() -> converts an int to float Math.random() -> generate a random number that is more than 0 but less than 1 randomNumberGenerator = Math.floor(Math.random() * 7) generateFourDigitOTP = Math.floor(Math.random() * 9000 + 1000)
